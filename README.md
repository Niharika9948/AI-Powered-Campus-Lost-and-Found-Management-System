
# AI-Powered Campus Lost and Found Management System

A web application to manage lost and found items on campus. Users can report lost items, upload found items, and let AI suggest possible matches.

---

## Features

- Report lost items with details and images  
- Upload found items  
- AI-based matching of lost and found items  
- Search and filter items  
- User login for students and staff  

---

## How to Run the Project

### Backend
1. Open the backend folder in your IDE or terminal.  
2. Make sure Python is installed on your system.  
3. Install required Python libraries manually (e.g., Flask, Django, or other dependencies).  
4. Run the backend server:

```bash
python app.py
````

5. The server will start at: `http://localhost:5000` (or your configured port).

### Frontend

1. Open the frontend folder in your code editor.
2. Make sure Node.js is installed on your system.
3. Install required dependencies manually (e.g., React libraries).
4. Start the frontend:

```bash
npm start
```

5. The app will open in your browser at: `http://localhost:3000` (or your configured port).

---

## Usage

* Open the frontend in your browser
* Register or log in as a user
* Report lost items or upload found items
* Check AI-based matches for your items
